# 🧬 AI Health Twin

A full-stack, real-time predictive healthcare engine designed to act as a digital twin for patient biomarkers. 

## 🚀 Features
* **Real-Time AI Predictions:** Adjust sliders and instantly see the Machine Learning model recalculate risk without reloading.
* **Futuristic UI/UX:** Built with a custom 3D glassmorphism interface, interactive glowing orbs, and debounced API requests.
* **Full-Stack Architecture:** Next.js (React/Tailwind) frontend communicating seamlessly with a Python (FastAPI/Scikit-Learn) backend.

## 🛠️ Tech Stack
* **Frontend:** Next.js, React, Tailwind CSS, Lucide Icons
* **Backend:** Python, FastAPI, Scikit-Learn, Pandas
